README – KI_Reifegrad

1. Voraussetzungen

- Windows-PC
- Internetverbindung für die Erstinstallation
- Python 3 (empfohlen: Version 3.10 oder höher)

1.1 Python installieren

1. Öffnen Sie die Seite https://www.python.org
2. Laden Sie den aktuellen Python-Installer für Windows herunter.
3. Starten Sie den Installer.
4. WICHTIG: Haken setzen bei „Add Python to PATH“.
5. Installation abschließen.

1.2 Projekt installieren

1. Die ZIP-Datei mit der Anwendung nach
   C:\KI_Reifegrad\
   (oder einen anderen Ordner Ihrer Wahl) entpacken.

2. Eingabeaufforderung (cmd) öffnen:
   - Windows-Taste drücken, „cmd“ eintippen, Eingabetaste.

3. In den Projektordner wechseln:
   cd C:\KI_Reifegrad

4. Optional: Virtuelle Umgebung anlegen und aktivieren:
   python -m venv venv
   call venv\Scripts\activate.bat

5. Benötigte Python-Pakete installieren:
   pip install -r requirements.txt

1.3 Anwendung starten

Nach der einmaligen Installation der Pakete reicht in Zukunft:

- Im Explorer den Ordner C:\KI_Reifegrad\ öffnen.
- Die Datei start_ki_reifegrad_app.bat per Doppelklick ausführen.

Die Web-App öffnet sich automatisch im Standard-Browser.
Standard-Adresse: http://localhost:8501
